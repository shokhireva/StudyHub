# StudyHub

StudyHub — учебная платформа с гибридной архитектурой: web + .NET backend + Tauri desktop shell.

## Структура

- `src/` — frontend (Vite + TS)
- `src-dotnet/` — backend (Clean Architecture: API / Application / Domain / Infrastructure)
- `src-tauri/` — desktop-оболочка на Tauri (Rust)

## Архитектура

Backend построен по принципам чистой архитектуры:
- Domain — бизнес-модели и правила
- Application — сценарии и логика
- Infrastructure — внешние зависимости
- API — HTTP слой

## Цель проекта
StudyHub представляет собой единую платформу для организации учебного процесса, объединяющую веб-клиент, серверную часть и десктопное приложение в рамках одной системы.

Основная задача системы — обеспечить централизованное управление учебными материалами, заданиями и взаимодействием пользователей через единый backend API с возможностью доступа из разных клиентских сред.

## Статус

Инициализация структуры и базовой связки модулей.
